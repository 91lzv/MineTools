# ⚒️ MineTools — The Ultimate Minecraft Toolkit
MineTools is a high-performance, community-driven Minecraft utility suite. Built with a focus on speed and a modern "cyber" aesthetic, it allows players to instantly look up accounts, check server statuses, and manage skins all in one place.

# ✨ Features
🔍 Advanced Search: Deep-search any Minecraft username or server IP directly from the hero section.

👤 Player Lookup: Fetch detailed profile information including UUIDs and player history.

🖥️ Server Status: Real-time server monitoring to check player counts, versions, and MOTDs.

🎭 Skin Stealer: Preview and download any player's skin in high resolution with pixel-perfect rendering.

📉 Live Metrics: Built-in real-time visitor and lookup tracking that persists across your sessions.

📱 Fully Responsive: A seamless experience across mobile, tablet, and desktop devices.

# 🎨 Design Philosophy
MineTools features a premium, dark-themed interface built for the modern web:

Dynamic Background: A custom animated grid system with a radial mouse-glow effect.

Fluid Animations: Smooth fadeInUpBlur transitions and interactive card glows.

Tailwind Powered: Clean, utility-first CSS for lightning-fast load times.

# 🛠️ Tech Stack
Frontend: HTML5, Tailwind CSS

Fonts: Inter (Body) & Oswald (Headings)

Icons: Lucide-style SVGs

Interactions: Native JavaScript
